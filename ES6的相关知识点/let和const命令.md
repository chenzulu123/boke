let 和 const 命令
1、let 是 ES6 用来声明变量的,let 声明的变量只在声明的代码块中有作用。
2、ES6 中具有代码块作用域,代码块作用域用{}区分

```js
{
}
```

```js
for (let i = 0; i < 5; i++) {
  console.log(i);
}
```
3、ES5中只有全局作用域和函数作用域
4、ES6在ES5的基础上新增了块级作用域
5、外部的块级作用域无法读取内部的块级作用域
```js
{
    let a = 2;
    {
        let b = 3;
        console.log(a);//2
    }
    console.log(b);//会报错，外部的块级作用域无法读取内部块级作用域的变量
}
```
